## Numbering rationale

- `00` = root (single file)
- `_meta` = vault-about-the-vault (leading underscore sorts first after numerics in most file managers)
- `01–09` = stable knowledge (system facts)
- `10` = UX/UI (separates code from craft)
- `11` = Simulation Mode (special demo concern)
- `12–13` = work + critique (changes more often)
- `14` = references (mostly external)
