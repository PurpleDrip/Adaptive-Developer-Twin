# Simulated Dashboard UI

Professional UI/UX instructions to create the ultimate "WOW" factor for investors and judges.

## Layout Styling & Interactions

1. **Monaco Terminal Simulator**:
   - Styling: Deep obsidian black backdrop (`hsl(224, 25%, 3%)`).
   - Feature: A typing blinking green cursor (`hsl(172, 66%, 50%)`).
   - Interaction: A dropdown to select languages (`Typescript`, `Python`, `Neo4j Cypher`).
2. **Telemetry Pipe Animation**:
   - Utilize SVG path animation representing telemetry packets flowing like binary particles from the Monaco editor toward a center "Server" node, which lights up when Fusion executes.
3. **Radar HUD Re-renders**:
   - The radar chart must not redraw flatly. Ensure smooth CSS ease animations when skill coordinates shift from e.g. mid `0.6` to lead `0.85`.
