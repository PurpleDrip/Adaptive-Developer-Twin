# Investor Pitch Mode Guide

Instructions for showcasing ADT to judges and clients to maximize engagement and demonstrate value.

## The 3-Minute Demo Script

1. **Minute 1: The Problem & The Solution**:
   - Introduce the "digital twin" concept.
   - Click "Start Simulation".
   - Show the simulated developer writing a complex AST parser in the mock IDE.
2. **Minute 2: The Core Tech (The "Black Box")**:
   - Direct attention to the telemetry pipeline stream.
   - Point out how raw keystrokes are translated into semantic embeddings via **CodeBERT**.
   - Show the live audit HUD populating with the cryptographic hashes (`SHA-HWID`).
3. **Minute 3: The Business Value**:
   - Show the bottom panel (Radar HUD).
   - Point out the instant update to the developer's Neo4j skill node.
   - Click "Allot Task" to show how the **CSA-Matching** engine recommends that developer based on their newly-updated digital twin profile.
