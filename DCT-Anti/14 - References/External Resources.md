# External Resources

Reference materials and libraries powering ADT:

1. **CodeBERT Transformer Pipeline**:
   - Model base: `microsoft/codebert-base`
   - Application: Semantic code analysis inside Fusion AI cores.
   - Reference: [HuggingFace Microsoft CodeBERT](https://huggingface.co/microsoft/codebert-base)

2. **Neo4j Graph Data Science (GDS)**:
   - Algorithms: PageRank, Closeness Centrality, Node Similarity.
   - Documentation: [Neo4j GDS Library Docs](https://neo4j.com/docs/graph-data-science/current/)

3. **FastAPI & Async IO**:
   - Application: All 9 microservices run on high-performance async FastAPI networks.
   - Documentation: [FastAPI Documentation](https://fastapi.tiangolo.com/)
